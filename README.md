# Meican Limited

Meican Limited is a leading construction chemicals supplier in Kenya, specializing in high-quality Sika products for various construction applications.

## About the Application

This is the official e-commerce website for Meican Limited, featuring:

### **Key Features**
- **Product Catalog**: Comprehensive showcase of construction chemicals organized by categories
- **Interactive Slideshow**: Dynamic product display with smooth animations and transitions
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Built with React, TypeScript, and Tailwind CSS
- **Performance Optimized**: Fast loading times and smooth user interactions

### **Product Categories**
- **Concrete Admixtures**: Water-reducing plasticizers and superplasticizers
- **Waterproofing**: Complete waterproofing systems for basements, roofs, and wet areas
- **Flooring Systems**: Industrial and commercial flooring solutions
- **Concrete Essentials**: Formwork release agents and reinforcement fibers
- **Elastic Sealing & Bonding**: High-performance sealants and adhesives
- **Tiling Products**: Professional tile adhesives and grouts
- **Grouts & Anchors**: High-strength grouts and anchoring systems
- **Injection Systems**: Structural injection and repair systems

### **Technology Stack**
- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS for utility-first styling
- **Animations**: Framer Motion for smooth transitions
- **Icons**: Lucide React for consistent iconography
- **Build Tool**: Vite for fast development and building
- **Package Manager**: npm for dependency management

### **Performance Features**
- **Image Optimization**: Local asset serving with proper compression
- **Code Splitting**: Optimized bundle sizes
- **Lazy Loading**: Efficient resource loading
- **Responsive Images**: Adaptive image serving

## Getting Started

### Prerequisites
- Node.js 16+ installed
- npm package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
